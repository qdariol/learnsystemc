# Learn SystemC fork
This repository is a fork of https://github.com/learnwithexamples/learnsystemc. The original tutorial is available in https://learnsystemc.com/basic/hello_world.

Here is what this fork adds:
- A `README.md` file in every sub-folder which contains the tutorial text from the main website. This way everything is available in the same place.
- Added `LICENSE` file for explicit statement.


# SystemC setup
Using docker: https://learnsystemc.com/setup/docker
I used SystemC-2.3.2 to run the examples in this tutorial.

# Tutorials
The tutorials are listed as sub-folders with a digital index of two numbers, and associated (short) title corresponding to the taught concept. All subfolders contain example source files, `Makefile` to build the example, and a `README.md` explaining the concept.
